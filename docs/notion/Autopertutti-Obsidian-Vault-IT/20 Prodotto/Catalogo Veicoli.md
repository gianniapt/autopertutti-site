# Catalogo Veicoli

Il catalogo deve essere separato per intento:

- auto in vendita
- auto a noleggio
- moto in vendita
- moto a noleggio

## Obiettivi del catalogo

- aiutare la scoperta
- mostrare filtri utili
- mantenere card pulite
- portare l'utente verso [[20 Prodotto/Scheda Veicolo]]

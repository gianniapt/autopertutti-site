# Conversion CTAs

Not all CTAs are equal. The CTA system should match business intent.

## Core CTAs

- request information
- request availability
- request a quote
- book a visit
- WhatsApp
- call now

## Why this matters

Each CTA should produce structured data for [[30 Systems/Lead Capture]] and eventually [[30 Systems/CRM and Lead Flow]].

## AI connection

When [[40 AI/AI Search]] or [[40 AI/AI Chat]] cannot fully answer, the user should be routed to a contextual CTA instead of a dead end.

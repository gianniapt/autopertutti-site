# Homepage Structure & Copy

Questa pagina definisce la struttura consigliata della homepage di Autopertutti, con testi, gerarchia dei contenuti e logica di posizionamento dei blocchi.

# Principio generale

La homepage deve presentare Autopertutti come una piattaforma automotive completa, senza dare priorità eccessiva a vendita, noleggio o servizi.

L’obiettivo è aiutare l’utente a scegliere rapidamente il percorso corretto:

- acquistare un’auto o una moto
- noleggiare un’auto o una moto
- prenotare un servizio di autolavaggio o officina
- parlare con il team

# Struttura consigliata

## 1. Header

Elementi principali:

- Logo Autopertutti
- Menu di navigazione
- Pulsante WhatsApp
- Pulsante “Contattaci”

Menu consigliato:

```
Auto
Moto
Noleggio
Servizi
Chi siamo
Contatti
```

Possibile variante più chiara:

```
Auto in vendita
Auto a noleggio
Moto
Servizi
Contatti
```

# 2. Hero section

La hero deve spiegare subito cosa fa Autopertutti, senza spingere solo un servizio.

## Headline proposta

```
Auto, moto, noleggio e servizi: tutto in un unico posto.
```

## Subheadline proposta

```
Trova il veicolo giusto, richiedi un noleggio o prenota un servizio per la tua auto con un team sempre disponibile.
```

## CTA principali

Le CTA devono essere bilanciate:

- Scopri i veicoli
- Trova un noleggio
- Prenota un servizio

## Layout consigliato

Desktop:

- sinistra: testo + CTA
- destra: immagine forte, collage veicoli o foto showroom/officina

Mobile:

- titolo
- testo breve
- tre pulsanti verticali
- immagine sotto

# 3. Selettore rapido esigenza

Subito dopo la hero, inserire un blocco con scelta rapida.

Titolo:

```
Di cosa hai bisogno oggi?
```

Card consigliate:

1. Voglio acquistare un veicolo
2. Voglio noleggiare un veicolo
3. Voglio prenotare un servizio
4. Voglio parlare con un consulente

Ogni card deve portare a una sezione o pagina specifica.

# 4. Macro-aree principali

Questa sezione deve mostrare le tre aree con peso visivo simile.

## Card 1 — Veicoli in vendita

Titolo:

```
Auto e moto in vendita
```

Testo:

```
Scopri i veicoli disponibili, confronta modelli, prezzi e caratteristiche, e richiedi informazioni al nostro team.
```

CTA:

```
Vedi veicoli in vendita
```

## Card 2 — Noleggio auto e moto

Titolo:

```
Noleggio auto e moto
```

Testo:

```
Trova il veicolo più adatto alle tue esigenze e verifica disponibilità, condizioni e prezzi di noleggio.
```

CTA:

```
Scopri il noleggio
```

## Card 3 — Autolavaggio e officina

Titolo:

```
Servizi per il tuo veicolo
```

Testo:

```
Prenota autolavaggio, diagnosi, manutenzione o assistenza con un servizio rapido e organizzato.
```

CTA:

```
Prenota un servizio
```

# 5. Veicoli in evidenza

Mostrare una selezione bilanciata:

- auto in vendita
- auto a noleggio
- moto in vendita / noleggio

Titolo:

```
Veicoli disponibili
```

Sottotitolo:

```
Una selezione aggiornata di auto e moto disponibili per vendita o noleggio.
```

Card veicolo:

- immagine
- marca e modello
- prezzo vendita o prezzo giornaliero
- stato disponibilità
- CTA: “Vedi dettagli”

# 6. Servizi in evidenza

Mostrare autolavaggio e officina in modo concreto.

Titolo:

```
Servizi per auto e moto
```

Card:

- Autolavaggio
- Diagnosi
- Manutenzione
- Officina

CTA:

```
Prenota appuntamento
```

# 7. Perché scegliere Autopertutti

Questa sezione serve per fiducia e credibilità.

Titolo:

```
Perché scegliere Autopertutti
```

Punti consigliati:

- Un unico punto per vendita, noleggio e servizi
- Team disponibile e consulenza personalizzata
- Veicoli selezionati e informazioni chiare
- Gestione rapida delle richieste
- Supporto via telefono, WhatsApp e chat

# 8. Come funziona

Sezione semplice in 3 step.

Titolo:

```
Come funziona
```

Step:

1. Scegli cosa ti serve
2. Invia la richiesta o contattaci
3. Ricevi supporto dal nostro team

# 9. Chat AI / assistente digitale

La chat può essere presentata come supporto rapido, non come sostituto del team.

Titolo:

```
Hai bisogno di aiuto nella scelta?
```

Testo:

```
Usa la chat per orientarti tra veicoli, noleggio e servizi. Se necessario, la tua richiesta verrà passata al nostro team.
```

CTA:

```
Apri la chat
```

# 10. CTA finale

Titolo:

```
Trova la soluzione giusta per te
```

Testo:

```
Che tu voglia acquistare, noleggiare o prenderti cura del tuo veicolo, Autopertutti è pronta ad aiutarti.
```

CTA:

- Contattaci
- Scrivici su WhatsApp
- Prenota un servizio

# 11. Footer

Contenuti:

- Logo
- Contatti

- Orari

# Note UX

- La homepage deve essere chiara entro 5 secondi.
- Non deve sembrare solo un autosalone, solo un noleggio o solo un’officina.
- Le tre macro-aree devono essere visivamente bilanciate.
- WhatsApp deve essere sempre accessibile.
- I form devono essere brevi.
- Su mobile, i pulsanti devono essere visibili e facili da cliccare.

# Obiettivo di conversione

La homepage deve portare l’utente verso una di queste azioni:

- vedere veicoli
- richiedere informazioni
- verificare disponibilità
- prenotare un servizio
- contattare il team
- avviare la chat

# 22 — Social proof e newsletter

# Social proof e newsletter

Questa pagina documenta due elementi da integrare nel sito Autopertutti: la sezione con clienti soddisfatti e il blocco newsletter.

# 1. Clienti soddisfatti / consegna chiavi

L’idea è mostrare foto reali di consegne, clienti soddisfatti e momenti di acquisto o noleggio completato.

# Obiettivo

Aumentare fiducia e credibilità mostrando che Autopertutti è un’attività reale, con clienti reali e consegne reali.

# Titoli possibili

```
Senza pensieri, dal primo clic alla consegna delle chiavi
```

```
Clienti soddisfatti, esperienze reali
```

```
Dalla scelta del veicolo alla consegna: ti seguiamo in ogni fase
```

# Layout consigliato

## Desktop

Carousel orizzontale con 3–4 immagini visibili.

```
[ Foto consegna 1 ] [ Foto consegna 2 ] [ Foto consegna 3 ] [ Foto consegna 4 ]
```

## Mobile

Carousel a singola card.

```
[ Foto cliente / consegna ]
← →
```

# Contenuti da usare

- Foto con clienti soddisfatti
- Foto consegna chiavi
- Foto veicolo consegnato
- Foto showroom / officina
- Brevi recensioni, se disponibili
- Nome cliente solo se autorizzato
- Città o zona solo se utile e autorizzato

# Nota privacy

Se nelle foto sono presenti persone riconoscibili, serve consenso esplicito all’uso dell’immagine.

Se il consenso non è disponibile, usare:

- foto veicolo senza volto
- dettagli delle chiavi
- immagini del momento consegna senza persone riconoscibili
- recensioni anonime

# 2. Newsletter

La newsletter serve a mantenere contatto con utenti non ancora pronti a comprare, noleggiare o prenotare un servizio.

# Obiettivo

Raccogliere contatti interessati a:

- nuovi veicoli disponibili
- offerte noleggio
- promozioni servizi
- consigli manutenzione
- contenuti del blog
- aggiornamenti dal mondo auto e moto

# Copy consigliato

Titolo:

```
Ricevi novità, offerte e consigli per il tuo veicolo
```

Alternativa:

```
Le novità più importanti sul mondo auto nella tua casella di posta
```

Sottotitolo:

```
Iscriviti per ricevere aggiornamenti su nuovi veicoli disponibili, offerte di noleggio, servizi e consigli utili per prenderti cura della tua auto o moto.
```

Campo form:

```
Indirizzo e-mail
```

CTA:

```
Iscriviti alla newsletter
```

# Segmentazione consigliata

Se possibile, chiedere interesse principale senza rendere il form troppo lungo.

Opzioni:

- Auto in vendita
- Noleggio
- Moto
- Officina
- Autolavaggio
- Tutte le novità

Form minimo consigliato:

```
Email
Interesse opzionale
Consenso marketing
```

# Collegamento con CRM / email marketing

Ogni iscrizione deve essere registrata con:

- email
- interesse principale
- fonte iscrizione
- pagina di provenienza
- consenso marketing
- data iscrizione

Tool possibili:

- HubSpot Marketing
- Brevo
- Mailchimp

# Tracking consigliato

Eventi:

```
newsletter_view
newsletter_signup
newsletter_interest_selected
newsletter_confirmed
```

UTM per campagne email:

```
utm_source=newsletter
utm_medium=email
utm_campaign=nome_campagna
```

# Regole importanti

- consenso marketing obbligatorio
- link Privacy Policy
- possibilità di disiscrizione
- double opt-in consigliato
- non inviare comunicazioni non richieste

# Frequenza consigliata

All’inizio:

```
1 newsletter ogni 2–4 settimane
```

Evitare invii troppo frequenti prima di avere contenuti reali e segmentazione.

# Raccomandazione finale

La sezione clienti soddisfatti serve a creare fiducia immediata.

La newsletter serve a mantenere relazione con utenti che non sono ancora pronti a convertire.

Entrambe devono essere tracciate e gestite con attenzione a consenso, privacy e qualità dei contenuti.

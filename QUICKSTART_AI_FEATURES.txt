ISSUE: Chat buttons don't appear
→ Check .env.local for typos
→ Restart dev server: npm run dev
→ Clear browser cache

ISSUE: Chat API returns 500 error
→ Verify OPENROUTER_API_KEY in .env.local
→ Check internet connection
→ Look at server logs: npm run dev output

ISSUE: Voice call not starting
→ Check VAPI keys in .env.local
→ Microphone permission denied? Check browser settings
→ VAPI requires HTTPS (won't work on localhost:3000)
→ Use Vercel deployment for testing voice calls

ISSUE: Lead not appearing in N8N
→ Check N8N_WEBHOOK_URL in .env.local
→ Verify N8N webhook is active (toggle on in UI)
→ Check N8N execution logs
→ Manually test: curl -X POST https://n8n-url/webhook/...

ISSUE: Build fails with TypeScript error
→ Check syntax in modified files
→ Verify imports are correct
→ Run: npm run build (see full error output)

WHERE LEADS GO:

Chat AI leads → /api/leads (POST) → N8N webhook → Airtable/Email
Voice Call leads → /api/leads (POST) → N8N webhook → Airtable/Email

Lead fields captured:
  • name (required)
  • phone (required)
  • email (fallback: non_fornita@chat.ai or non_fornita@voice.ai)
  • message (optional)
  • service ("ai_chat" or "voice_call")
  • timestamp (ISO 8601)
  • channel ("Chat AI" or "Voice Call VAPI")
